# ⚙️ GearStack - Inventory Management System

GearStack es una aplicación web robusta construida con **Django** para la gestión y control de inventario de herramientas y equipos. Este proyecto sirve como un MVP funcional que demuestra el flujo completo de datos (MVT), manejo de entornos virtuales modernos y diseño de interfaz de usuario.

## 🚀 Características actuales
- **Gestión de Inventario:** Visualización completa de ítems con categorización.
- **Navegación Dinámica:** Vistas de detalle individuales para cada herramienta mediante parámetros de URL.
- **Base de Datos Relacional:** Uso de modelos vinculados (Items <-> Categories).
- **Datos Sintéticos:** Script de "seeding" integrado para pruebas masivas con datos realistas.
- **Interfaz Profesional:** Diseño responsivo en Dark Mode utilizando Bootstrap 5.

## 🛠️ Stack Tecnológico
- **Backend:** Python 3.12 + Django 5.x
- **Gestión de Paquetes:** [uv](https://github.com/astral-sh/uv) (Extremadamente rápido)
- **Base de Datos:** SQLite (Desarrollo)
- **Frontend:** HTML5, CSS3, Bootstrap 5
- **Entorno:** WSL2 (Ubuntu)
